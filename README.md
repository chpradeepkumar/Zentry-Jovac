# Zentry AI Platform - Frontend Prototype

A premium, fully responsive 4-page digital product website designed to showcase a modern AI workflow automation startup.

## 🚀 Key Deliverables Included
- **Home Page (`index.html`):** Hero section, 4 dynamic feature nodes, and strategic business benefit matrix.
- **Product Page (`product.html`):** Detailed framework mechanics, 3-tier adaptive pricing grid, client case-studies, and integrated FAQ block.
- **Contact Page (`contact.html`):** Comprehensive secure user registration module featuring advanced HTML5 constraint validations (regex phone verification, data clears).
- **About Page (`about.html`):** Tactical overview of company genesis and operational infrastructure roles.
- **Global Engine Styling (`style.css`):** Fluid grid system built natively on CSS Flexbox & Grid utilizing an dark-indigo visual layout theme.
